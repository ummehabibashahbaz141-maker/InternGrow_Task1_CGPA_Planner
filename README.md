# InternGrow_Task1_CGPA_Planner
CGPA Calculator C++ Project - InternGrow Task 1

📌 Automated Semester CGPA Planner
👩‍💻 Student Information System (C++ Project)

This project is an Automated Semester CGPA Planner developed in C++.
It is designed to help students calculate their semester GPA and overall CGPA based on their courses, credit hours, and grades.

🎯 Project Purpose

The main goal of this project is to simulate a real academic grading system where a student can:

Enter multiple semesters
Add courses for each semester
Assign credit hours and grades
Automatically calculate GPA and CGPA
View a complete academic result report
⚙️ Features

✔ Multiple semester input system
✔ Course-wise credit hour management
✔ Grade to grade-point conversion system
✔ Semester GPA calculation
✔ Final CGPA calculation
✔ CGPA Prediction feature (target-based analysis)
✔ Clean tabular result output

🧠 Logic Used
Conditional statements (if-else) for grade conversion
Loops for semester & course handling
Vectors for dynamic data storage
Functions for modular grade conversion
Mathematical GPA & CGPA calculation formulas

📌 Upgrade Feature (Important)

This project also includes a CGPA Prediction System, which allows students to:

👉 Enter target CGPA
👉 Enter remaining credit hours
👉 Check if target CGPA is achievable or not

This helps students plan their future academic performance.
