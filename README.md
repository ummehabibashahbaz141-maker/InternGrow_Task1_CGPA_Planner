### 👩‍💻 InternGrow Internship Project – Task 1 (C++)

## 🎯 Project Description

This project is an Automated Semester CGPA Planner developed in C++.  
It calculates semester GPA and final CGPA based on courses, credit hours, and grades.

---

## ⚙️ Features

- Multiple semester input system  
- Course-wise credit hour handling  
- Grade to grade-point conversion  
- Automatic Semester GPA calculation  
- Final CGPA calculation  
- CGPA Prediction system (target-based analysis)  
- Clean tabular result output  

---

## 🧠 Concepts Used

- Loops (for nested input handling)  
- Functions (grade conversion logic)  
- Vectors (dynamic data storage)  
- Conditional statements (if-else logic)  
- GPA & CGPA formulas
